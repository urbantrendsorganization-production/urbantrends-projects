## Headline

**Paper-free client onboarding and KYC for Kenyan MFIs and agencies.**

## Subhead

OnboardKit takes your field agents from a client's ID and phone to an approved, audit-ready record — captured on a phone, reviewed on a desktop, done in minutes instead of days.

## Features

- **Field-agent mobile onboarding.** Agents enrol clients on an Android app: client details, KYC documents, phone verification, and consent — with every step saved as they go, so a dropped connection never loses work.
- **Reviewer desktop queue.** Reviewers see submitted applications in a branch queue, view the form data side-by-side with the captured documents, and approve, reject with a reason, or return for correction.
- **KYC document capture.** ID front, ID back, selfie, and proof of address — captured with the camera, compressed on-device, and stored securely with short-lived access links. National ID and KRA PIN captured on the client record.
- **Compliance and audit trail.** Every status change is written to an append-only event log, giving each client a complete, tamper-evident onboarding history for your compliance team.

## How it works

1. **Capture.** A field agent onboards the client on the phone — details, KYC documents, a one-time code to the client's own +254 number, and digital consent.
2. **Review.** A reviewer checks the application on the desktop, against the actual documents, and approves it.
3. **Confirm.** The client is assigned a permanent client number and receives an approval SMS — and your team can export approved clients to CSV or Excel.
